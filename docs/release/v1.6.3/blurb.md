**Krust Studio v1.6.3** - patch release.

Fixes macOS unsigned builds so they can launch after Gatekeeper bypass, and keeps `DATE` / `DATETIME` grid values shown as database-style literals instead of UTC ISO strings.

Package: `krust-studio-app-1.6.3-setup.exe` for Windows, `krust-studio-app-1.6.3.dmg` for Apple Silicon macOS.
