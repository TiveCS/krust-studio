Patch release adding dialect-aware SQL formatting and display-only DDL
prettification.

## Added

- **Format SQL** — format query-editor text with the active MySQL, PostgreSQL, or
  SQLite dialect using the toolbar or `Shift+Alt+F`.
- **Pretty DDL** — format Structure DDL and generated DDL review for display
  without changing copied, executable, captured, or exported SQL.

## Install

Grab the `1.6.6` installer for your OS below. Windows SmartScreen and macOS
Gatekeeper may warn because these builds are not publicly notarized/code-signed.
