**Krust Studio v1.6.4** - patch release.

Fixes the macOS CI packaging path for unsigned builds and adds native calendar/date-time inputs when editing `DATE`, `DATETIME`, or timestamp-like cells in the table data grid.

Package: `krust-studio-app-1.6.4-setup.exe` for Windows, `krust-studio-app-1.6.4.dmg` for Apple Silicon macOS.
