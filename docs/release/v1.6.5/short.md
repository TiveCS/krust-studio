Patch release - open and save `.sql` files from the SQL editor.

## Added

- **Open / save `.sql` files** - open a `.sql` file into a new query tab (tab bar **Open SQL file…** or tab right-click) and save the editor's SQL back out with **Save .sql**. Opening is a one-shot import: the filename shows as the tab title, but the tab is not linked to the file.

## Install

Grab the `1.6.5` installer for your OS below. Windows SmartScreen and macOS Gatekeeper may warn because these builds are not publicly notarized/code-signed.
