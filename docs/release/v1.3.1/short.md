A small fix-and-polish patch on top of v1.3.0.

## What's new
- **🐛 Structure editor scroll** — wide tables no longer overflow behind the commit bar; the column list scrolls in its own area.
- **🔧 Add column moved to the footer** — bottom-left next to Commit / Discard, no more scrolling to the end of the list; adding clears the active column filter.
- **🔄 Check for updates** — manual check in the title-bar app menu (**Krust Studio ▾ → Check for updates**).

## 📦 Install
Grab `krust-studio-app-1.3.1-setup.exe` below. SmartScreen may warn (unsigned) → **More info → Run anyway**. Existing installs auto-update.

> ⚠️ Use at your own risk — keep backups before editing real data.
