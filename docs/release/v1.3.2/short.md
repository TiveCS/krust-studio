Hotfix for the title-bar menu being hidden behind the sidebar.

## What's new
- **🐛 Title bar no longer hidden** — the sidebar was anchored to the viewport top (`fixed inset-y-0`), covering the "Krust Studio ▾" app menu added in v1.3.1. It now starts below the title bar so **Check for updates** and any future menu items are accessible.

## 📦 Install
Grab `krust-studio-app-1.3.2-setup.exe` below. SmartScreen may warn (unsigned) → **More info → Run anyway**. Existing installs auto-update.

> ⚠️ Use at your own risk — keep backups before editing real data.
