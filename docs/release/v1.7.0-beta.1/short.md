# Krust Studio 1.7.0-beta.1

First 1.7.0 pre-release. **Opt in via Settings → Updates → "Receive beta updates".**

- **Redis** — native key browser + staged editor (strings, hashes, lists, sets,
  sorted sets, streams), TTL, binary values, WATCH/MULTI commit, Redis Mutation
  history. Search loaded keys in the command palette (Ctrl/⌘+P).
- **Procedures & Functions** — browse, view definition, execute (typed params,
  OUT/INOUT), and create/drop on MySQL/MariaDB + PostgreSQL. Routine Execution
  history stream. (Editing an existing MySQL routine lands in a later beta.)
- **Data grid** — keyboard navigation (arrows, Tab, Home/End, PageUp/Down),
  Enter/F2 to edit, type-to-edit, Ctrl/⌘+A select-all, and client-side **Find**
  (Ctrl/⌘+F) over the loaded page.
- **Settings** — configurable Find shortcut, toast position, and beta channel.
- Vertical mouse-wheel scrolls the tab strip horizontally.

Beta feedback: see the 1.7.0-beta tracking issue.
